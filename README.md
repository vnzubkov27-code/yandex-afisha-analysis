Анализ лояльности пользователей Яндекс Афиши
Цель проекта
Провести исследовательский анализ лояльности пользователей Яндекс Афиши для выявления факторов, влияющих на повторные покупки.

Основные задачи:
Загрузите данные и проведите их первичную проверку.
Провести предварительную обработку данных.
Провести исследовательский анализ данных:
Проанализируйте воронку продаж от первого визита до покупки.
Рассчитайте и проанализируйте ключевые показатели лояльности.
Провести сегментацию пользователей.
Изучить, как различные характеристики пользователей (источник привлечения, география, поведение) влияют на их лояльность.
Сформулировать выводы и практические рекомендации для отдела маркетинга по повышению лояльности пользователей.
