# Анализ лояльности пользователей Яндекс Афиши

## Цель проекта
Провести исследовательский анализ лояльности пользователей Яндекс Афиши для выявления факторов, влияющих на повторные покупки.

## Основные задачи

### 1. Загрузка и первичная проверка данных
- Выгрузка данных из базы данных SQL
- Изучение общей информации о данных
- Проверка корректности выгрузки и объема данных

### 2. Предварительная обработка данных
- Конвертация валюты (тенге → рубли)
- Обработка пропущенных значений
- Преобразование типов данных
- Проверка и обработка выбросов
- Нормализация категориальных данных

### 3. Исследовательский анализ данных
- Анализ воронки продаж от первого визита до покупки
- Расчет и анализ ключевых показателей лояльности
- Сегментация пользователей
- Изучение влияния характеристик пользователей на лояльность:
  - Источник привлечения
  - География
  - Поведенческие паттерны

### 4. Формулировка выводов и рекомендаций
- Подготовка выводов по результатам анализа
- Разработка практических рекомендаций для отдела маркетинга
- Предложения по повышению лояльности пользователей

## Ключевые метрики анализа
- Коэффициент возврата пользователей
- Средняя выручка с заказа
- Среднее количество билетов в заказе
- Время между заказами
- Распределение пользователей по сегментам лояльности

## Технологии
- Python
- Pandas
- SQL
- Matplotlib/Seaborn
- Phik (корреляционный анализ)
